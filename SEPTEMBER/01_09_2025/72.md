## Edit Distance
**Understand**
Given two strings `word1` and `word2` return the minimum number of oprations required to convert `word1` to `word2`.
You can perform these operations:
- Insert a character
- Delete a character
- Replace a character

**Example**
Input: `word1` = `horse`, `word2` = `ros`
Output: 3
Explanation: 
horse -> rorse (replace 'h' with 'r')
rorse -> rose (remove 'r')
rose -> ros (remove 'e')
